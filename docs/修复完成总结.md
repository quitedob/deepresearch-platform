## 二、功能说明

### 1. 网络搜索功能

**使用方式**：
```javascript
// 前端调用
await chat({
  session_id: 'xxx',
  message: '最新的AI新闻',
  use_web_search: true  // 启用网络搜索
})
```

**后端处理**：
- 检测到 `use_web_search=true` 时，在消息前添加 `[使用网络搜索]` 标记
- 如果使用智谱 AI（glm-4.6），会自动调用其 web_search 工具
- 其他模型会通过 eino 的 WebSearchTool 进行搜索

**实现位置**：
- 前端：`vue/src/api/chat.js`
- 后端：`src/internal/api/v1/chat.go`
- 工具：`src/internal/eino/tool/websearch.go`

### 2. 深度思考功能

**使用方式**：
```javascript
// 前端调用
await chat({
  session_id: 'xxx',
  message: '解释量子纠缠的原理',
  use_deep_think: true  // 启用深度思考
})
```

**后端处理**：
- 检测到 `use_deep_think=true` 时，自动切换到深度思考模型
- 根据当前提供商选择对应的深度思考模型：
  - DeepSeek → deepseek-reasoner
  - 智谱 AI → glm-4.6
  - Ollama → gemma3:12b

**实现位置**：
- 前端：`vue/src/api/chat.js`
- 后端：`src/internal/api/v1/chat.go` 的 `getDeepThinkingModel()` 方法

### 3. 深度研究功能

**使用方式**：
```javascript
// 前端调用研究 API
await researchAPI.startResearch({
  query: '人工智能的最新进展',
  research_type: 'deep'
})
```

**后端处理**：
- 使用完整的 eino 研究链条
- 包含 Agent + Tools（WebSearch, ArXiv, Wikipedia）
- 多步骤研究流程：planning → executing → synthesis → completed

**实现位置**：
- 前端：`vue/src/api/research.js`
- 后端：`src/internal/api/v1/research.go`
- 核心：`src/internal/eino/agent/research.go`

## 三、配置说明

### 1. 模型配置（config.yaml）

所有模型从 `config.yaml` 读取，不要硬编码：

```yaml
llm:
  default_provider: deepseek
  providers:
    deepseek:
      models:
        - deepseek-chat      # 默认模型
        - deepseek-reasoner  # 深度思考模型
      default_model: deepseek-chat
    zhipu:
      models:
        - glm-4.6           # 深度思考模型（支持 web_search）
        - glm-4.5-air       # 默认模型
      default_model: glm-4.5-air
    ollama:
      models:
        - gemma3:4b         # 默认模型
        - gemma3:12b        # 深度思考模型
        - qwen3:8b          # 默认模型
      default_model: gemma3:4b
```

### 2. 环境变量

需要配置的 API Key：
```bash
# 智谱 AI（用于网络搜索）
ZHIPU_API_KEY=your_zhipu_api_key

# DeepSeek
DEEPSEEK_API_KEY=your_deepseek_api_key
```

## 六、前端使用示例

```javascript

})
```

### 2. 聊天功能
```javascript
import { chat, chatStream } from '@/api/chat'

// 普通聊天
const response = await chat({
  session_id: sessionId,
  message: '你好'
})

// 网络搜索聊天
const response = await chat({
  session_id: sessionId,
  message: '最新的AI新闻',
  use_web_search: true
})

// 深度思考聊天
const response = await chat({
  session_id: sessionId,
  message: '解释量子纠缠',
  use_deep_think: true

// 流式聊天
await chatStream(
  {
    session_id: sessionId,
    message: '写一篇文章',
    use_deep_think: true
  },
  (content) => console.log('收到:', content),
  (error) => console.error('错误:', error),

)
```
