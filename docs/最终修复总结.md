## 功能使用说明

### 1. 用户注册
```bash
curl -X POST http://localhost:8080/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "email": "test@example.com",
    "password": "password123",
    "full_name": "Test User"
  }'
```

### 2. 用户登录
```bash
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "password": "password123"
  }'
```

### 3. 普通聊天
```bash
curl -X POST http://localhost:8080/api/v1/chat/chat \
  -H "Authorization: Bearer YOUR_TOKEN" \

  -d '{
    "session_id": "xxx",
    "message": "你好"
  }'
```

## 前端使用示例

### 注册登录
```javascript
import { register, login } from '@/api/user'

// 注册
await register({
  username: 'testuser',
  email: 'test@example.com',
  password: 'password123',
  full_name: 'Test User'
})

// 登录
await login({
  username: 'testuser',
  password: 'password123'
})
```

### 聊天功能
```javascript
import { chat, chatStream } from '@/api/chat'

// 普通聊天
await chat({
  session_id: sessionId,
  message: '你好'
})

// 网络搜索
await chat({
  session_id: sessionId,
  message: '最新AI新闻',
  use_web_search: true
})

// 深度思考
await chat({
  session_id: sessionId,
  message: '解释量子纠缠',
  use_deep_think: true
})

// 流式聊天
await chatStream(
  {
    session_id: sessionId,
    message: '写一篇文章',
    use_deep_think: true
  },
  (content) => console.log(content),
  (error) => console.error(error),
  () => console.log('完成')
)
```

### 模型管理
```javascript
import { 
  getModels, 
  isDeepThinkingModel, 
  getDeepThinkingModelByProvider 
} from '@/api/model'

// 获取所有模型
const models = await getModels()

// 检查是否为深度思考模型
if (isDeepThinkingModel('deepseek-reasoner')) {
  console.log('这是深度思考模型')
}

// 获取提供商的深度思考模型
const model = getDeepThinkingModelByProvider('deepseek')
// 返回: 'deepseek-reasoner'
```
