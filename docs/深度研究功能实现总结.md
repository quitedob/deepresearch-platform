## 当前问题

⚠️ **研究执行卡住**

**症状：**
- 研究任务可以成功启动
- 状态一直停留在 `executing` (progress: 0.1)
- 没有进度更新
- 没有错误日志

**可能原因：**

1. **Agent 执行超时或卡住**
   - ReAct 循环可能在等待 LLM 响应
   - 工具调用可能失败但没有正确处理错误

2. **EventStream 没有正确推送**
   - 进度回调可能没有被触发
   - SSE 连接可能有问题

3. **LLM 调用问题**
   - 智谱AI API 调用可能失败
   - 模型响应格式可能不符合预期
